# Python fronend

Simple app written with `flask`.  It makes a request to the backend and display value on a html template.

It requires a respond from backend with the following json format.

```json
{
  "messages": {
    "name": "test me",
    "email": "email",
    "phone": "phone",
    "address": "address"
  }
}
```
